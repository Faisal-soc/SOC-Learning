# SOC Notes

## What is SIEM?
SIEM is a security tool used to collect and analyze logs from systems and devices.

## What is SOC?
SOC stands for Security Operations Center.
It is responsible for monitoring and responding to security incidents.

## Common Alerts
- Failed login attempts
- Malware detection
- Suspicious IP activity
- Multiple authentication failures

## Tools Used in SOC
- Splunk
- Wireshark
- EDR
- IDS/IPS
