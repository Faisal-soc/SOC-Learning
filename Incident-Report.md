# Incident Report

## Alert
Multiple failed login attempts detected from a suspicious IP address.

## Investigation
The investigation showed repeated failed authentication attempts against a user account.

## Findings
- Source IP: 192.168.1.50
- Multiple failed logins detected
- Possible brute force attack activity

## Action Taken
- Blocked the suspicious IP address
- Reset the affected user password
- Escalated the alert to SOC team

## Conclusion
The activity was identified as suspicious and handled according to SOC procedures.
